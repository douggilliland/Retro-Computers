EASy68K Introduction

Editor, Assembler, Simulator and S-Record/Binary file utility for the 68000 microprocessor.

This package contains everything you need to create and run 68000 assembly language programs on a Windows PC. All programs and files are distributed under the GNU general public use license.

Please see the help files for full contact information, bug reports, downloading and credits.

Several example programs are included that demonstrate various features of EASy68k. These programs may be found in the 'Examples' folder.

The latest version of EASy68k may be found at: http://www.easy68k.com


Enjoy.
