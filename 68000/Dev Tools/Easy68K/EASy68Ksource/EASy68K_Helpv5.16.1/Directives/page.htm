Assembler Directives - PAGE

PAGE - The page directive writes the NEW_PAGE_MARKER to the list (.L68) file if the list file is created during assembly. The page directive produces no object code. No label is permitted and any comments are ignored. If the list file is printed from the EASy68K editor a new page command will be sent to the print device each time a NEW_PAGE_MARKER is printed. The NEW_PAGE_MARKER is not printed. The NEW_PAGE_MARKER will appear in the list file as the text string "<------------------------------ PAGE ------------------------------>"

Usage:
       PAGE
